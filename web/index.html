Anleitung / Instructions

Note: Java 7 muss installiert sein. You need to have Java 7 installed on your system.

  1. Klicken Sie auf den Launch-Knopf.
    Click the Launch button above.
  2. Java Webstart wird aktiviert. Beim ersten Start sowie bei der Verfügbarkeit von Updaten werden der Editor und alle Komponenten automatisch Ihren Rechner übertragen.
    Java Webstart will launch. If this is your first time using the Editor (or if there are updates available) the Editor and all required components will be automatically transferred to your computer.
  3. Der Editor ist derzeit mit einem selbstsignierten Zertifikat versehen. Daher erscheint bei jedem Start ein Warnhinweis. Es ist jedoch vollkommen sicher, den Editor zu benutzen. Setzen Sie das Häkchen und klicken Sie anschließend auf den Ausführen-Button (siehe Bild rechts).
    The Editor is currently using a self-signed certificate. This will cause a warning message to appear every time you launch the Editor. It is however perfectly safe to use the software. Just check the checkbox and click on the Launch button (see image on the right).
  4. Der Editor startet, das Login-Fenster erscheint.
    The Editor will start and the login window appears.
